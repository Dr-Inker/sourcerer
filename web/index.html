Sourcerer

An AI technical-sourcing agent. It finds an engineer, researches them from public sources, and writes a grounded, cited fit-brief — every claim must point at real evidence, or it doesn't get made.

Pick a role. Watch it run discover → research → synthesize, then read the brief — with the grounding score, clickable citations, and the “unverified” list of things it refused to assert. Source on GitHub ↗
